📖 Setup Guide — How to create a Test Sheet ▼
  1. Open your live Google Sheet:
    https://docs.google.com/spreadsheets/d/1_pbyhKL1IElgneBZHFIWG76Cv6hNKdUw0fgSHQjMN6M
  2. Click File → Make a copy and name it "DCFS Memorial Tracker — TEST"
  3. In the copy, go to Extensions → Apps Script
  4. Copy the script from your live deployment (or paste the same Code.gs contents)
  5. Click Deploy → New deployment → Web app
  6. Set Execute as: Me and Who has access: Anyone
  7. Click Deploy and copy the URL that appears
  8. Paste that URL into the Test URL field above
